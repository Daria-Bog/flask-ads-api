# Aiohttp Ads API (v2: Secure & Validated)

Асинхронный REST API сервис для площадки объявлений. Проект прошел рефакторинг: с Flask на **aiohttp**, внедрена система безопасности JWT и валидация данных.

## Что было исправлено (согласно замечаниям)
* **JWT Authentication:** Вместо передачи ID пользователя реализована полноценная система токенов с использованием библиотеки `PyJWT`. Токены имеют срок жизни и проверяются через `auth_middleware`.
* **Валидация через Pydantic:** Все входящие данные (регистрация, создание и обновление объявлений) проходят строгую проверку схем. При ошибках клиент получает статус `400` с описанием.
* **Метод PATCH:** Добавлена возможность частичного обновления объявлений с обязательной проверкой прав владельца (403 Forbidden для чужих записей).
* **Архитектура:** Использование Middleware для централизованного управления сессиями БД и аутентификацией.

## Технологический стек
* **Python 3.11**
* **aiohttp** — асинхронный сервер.
* **SQLAlchemy 2.0 (Async) + asyncpg** — работа с БД.
* **Pydantic** — валидация данных.
* **PyJWT** — безопасность и токены.
* **Bcrypt** — хеширование паролей.
* **Docker & Docker Compose** — контейнеризация.
