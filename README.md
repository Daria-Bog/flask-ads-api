# Aiohttp Ads API

Асинхронный REST API сервис для площадки объявлений, переписанный с Flask на **aiohttp**. Проект включает в себя систему аутентификации, работу с базой данных PostgreSQL и полную докеризацию.

## Основные изменения (переход с Flask)
* **Асинхронность:** Вместо блокирующего Flask использован `aiohttp.web`, что позволяет обрабатывать тысячи соединений в одном потоке.
* **ORM:** Синхронная SQLAlchemy заменена на асинхронную версию с драйвером `asyncpg`.
* **Архитектура:** Использование `Class Based Views` для логической группировки методов GET, POST, DELETE.
* **Middlewares:** Сессии базы данных управляются через промежуточное ПО (middleware) для автоматического закрытия соединений.

## Технологический стек
* **Python 3.11**
* **aiohttp** (сервер)
* **SQLAlchemy 2.0 (Async)** (работа с БД)
* **PostgreSQL** (хранилище)
* **Bcrypt** (хеширование паролей)
* **Docker & Docker Compose** (контейнеризация)

## Функционал
1. **Пользователи:** Регистрация и аутентификация (выдача токена на основе ID).
2. **Объявления:**
   - Создание (только для авторизованных).
   - Удаление (только для владельца объявления).
   - Получение по ID (публично).
3. **Безопасность:** Пароли хранятся в виде защищенных хешей.
